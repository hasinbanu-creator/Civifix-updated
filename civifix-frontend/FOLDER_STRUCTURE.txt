🚀 QUICK START

1. Install dependencies:
   npm install

2. Configure environment:
   cp .env.example .env
   # Edit .env with backend API URL

3. Start development:
   npm start

4. Run on device:
   • Expo Go: Scan QR code
   • iOS: Press 'i'
   • Android: Press 'a'
   • Web: Press 'w'
